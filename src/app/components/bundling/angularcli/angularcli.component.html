What is Angular?

Angular is a platform that makes it easy to build applications with the web. Angular combines declarative templates, dependency injection, end to end tooling, and integrated best practices to solve development challenges.

Angular is a framework for building client applications in HTML and either JavaScript or a language like TypeScript that compiles to JavaScript. The framework consists of several libraries, some of them core and some optional.

What is Angular CLI?

The Angular CLI is a tool to initialize, develop, scaffold and maintain Angular applications

How do you intall Angular CLI?

npm install -g @angular/cli

How do you create and run a project?

ng new my-project
cd my-project
ng serve
