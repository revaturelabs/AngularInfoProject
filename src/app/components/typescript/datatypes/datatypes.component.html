Data Types in TypeScript

Data Type Documentation

Boolean

  • This is the most basic data type in TypeScript.
  • Booleans can only be assigned as true/false.
  • Example:
    • let thisBoolean: boolean = false;

Number

  • All numbers in TypeScript are floating point values.
  • TypeScript numbers supports decimals, hexidecimals, binary, and octal literals.
  • Examples:
    • Decimal: let decimal: number = 13;
    • Hexidecimals: let hexi: number = 0xf00d;
    • Binary: let bin: number = 0b1010;
    • Octal: let oct: number = 0o744;

String

  • Strings consist of textual data.
  • When assigning a string, you can use either ' ' or " ".
  • Examples:
    • let name: string = 'Delli';
    • let name: string = "Delli";

Arrays

  • Arrays in TypeScript work similarly to those found in JavaScript.
  • Arrays can be written in two different ways:
  • Examples:
    • let arrayOne: number[] = [13, 13, 13, 13];
    • let arrayTwo: Array < number> = [13, 13, 13, 13];

Enums

  • A way of giving numbers a friendly name.
  • By default, enums begin numbering their members at 0.
  • You can assign an enum member to a specific number.
  • Examples:
    • enum Colors {{ "{RED, GREEN, BLUE}" }}
      let c: Colors = Colors.GREEN;
    • enum Colors {{ "{RED = 1, GREEN, BLUE}" }}
      let c: Colors = Colors.GREEN;
    • enum Colors {{ "{RED = 1, GREEN = 2, BLUE = 5}" }}
      let c: Colors = Colors.GREEN;

Any

  • When you don't know what the data type will be; use the "any" data type.
  • An "any" data type can be assigned to any of the data types available in TypeScript.
  • This data type can be changed to any other data type at any time.
  • Examples:
    • let unSure: any = 13;
      any = "maybe this is a string";
      any = false;

Void

  • Void is basically the reverse of any.
  • You usually use this data type for functions that return no other data type.
  • You can assign a variable to void. If you do this, you can only assign that variable to "null" or "undefined".
  • Examples:
    • function doNothing(): void {{ "{ alert(This function has no return value.); }" }}
    • let dontUseThisVariable: void = undefined;
